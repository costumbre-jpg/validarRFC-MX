# Costos operativos estimados

Estos costos ayudan a justificar el precio de venta mostrando el costo real de operar la plataforma.

## Infraestructura base (estimado mensual)

- **Vercel Pro**: ~$20 USD
- **Supabase**: ~$25–$100 USD (según tráfico y DB)
- **Stripe**: comisión por transacción (sin fee fijo)
- **Resend**: ~$0–$20 USD (según volumen de emails)
- **Upstash Redis** (opcional): ~$0–$20 USD

## Costos variables

- Tráfico y validaciones reales (SAT externo)
- Volumen de emails
- Storage (logos/avatars)

## Notas clave para el comprador

- El costo se escala con usuarios pagados.
- La infraestructura actual permite crecer sin re-arquitectura inmediata.
- El margen es alto si la adquisición de clientes es eficiente.
