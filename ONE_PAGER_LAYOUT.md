# One‑pager (diseño y layout)

Formato recomendado: 1 página vertical.

## Estructura
1. Header con logo + título
2. Problema (2–3 líneas)
3. Solución (2–3 líneas)
4. Features clave (5 bullets)
5. Modelo de negocio
6. Estado del producto
7. Contacto

## Diseño
- Fondo blanco
- Títulos en negrita
- Íconos simples para features
- Footer con correos
