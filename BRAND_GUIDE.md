# Guía rápida de marca (Maflipp)

## Logo
- Archivo: `public/Maflipp-logo.png`
- Uso recomendado: fondo blanco, tamaño mínimo 28px

## Tipografías sugeridas
- Títulos: Inter / Poppins / Helvetica
- Texto: Inter / Arial

## Colores
- Primario: usar color principal de Maflipp
- Secundario: derivado del primario en 60–70% intensidad
- Neutros: gris claro para fondos, negro suave para texto

## Tono
Corporativo, claro y directo.
