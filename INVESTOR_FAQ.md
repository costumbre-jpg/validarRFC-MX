# FAQ Comercial (inversionista/comprador)

## ¿Qué vende Maflipp?
Una plataforma B2B de validación de RFC con dashboard, analytics, API pública y white label.

## ¿Está listo para producción?
Sí. El core es funcional; solo requiere configurar credenciales de servicios externos.

## ¿Qué incluye la venta?
Código fuente completo, migraciones, documentación, y handoff operativo.

## ¿Qué tan escalable es?
La arquitectura (Next.js + Supabase + Vercel) permite escalar sin rediseños.

## ¿Cómo genera ingresos?
Suscripción mensual por volumen (Free / Pro / Business).

## ¿Qué riesgos existen?
Dependencia de SAT (servicio externo) y costos variables por volumen.

## ¿Qué mejora el valor?
Clientes activos, métricas de uso (MRR) y casos de éxito.
